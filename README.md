# JavaScript Practice Repository

Welcome! 👋  
This repository contains **JavaScript practice exercises** that cover fundamental concepts and methods for beginners and intermediate learners.

## 🔹 Contents
- **for.js** – examples of `for` loops
- **arrow function.js** – arrow function practice
- **even.js** – check even numbers
- **every method.js** – using `every()` array method
- **fill.js** – using `fill()` array method
- **filter.js** – using `filter()` array method
- **objects.js** – working with objects
- **slice.js** – using `slice()` method
- **splice.js** – using `splice()` method

## 🔹 Purpose
- Strengthen understanding of JavaScript fundamentals.
- Practice array methods, loops, and object manipulation.
- Prepare for coding interviews and projects.

## 🔹 Benefits
- Improve problem-solving skills in JavaScript.
- Learn practical coding techniques for real-world applications.
- Build a strong foundation for frontend development with JavaScript.

> “Practice makes perfect — these exercises are meant to help you master JavaScript fundamentals.” 🚀
